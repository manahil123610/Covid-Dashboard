# Covid-Dashboard
A web-based dashboard built with Streamlit for exploring, analyzing, and visualizing survey and performance data.
## Project Structure

dashboard_project/
├── data/
│ └── 1780429023919_data.csv
├── dashboard.py
├── requirements.txt
└── README.md

## Features

* Interactive visualizations
* Data filtering options
* Summary statistics
* User-friendly dashboard interface

## Technologies Used

* Python
* Streamlit
* Pandas
* NumPy
* Matplotlib
* Plotly

## Running the Application

1. Install dependencies:

pip install -r requirements.txt

2. Run the application:

streamlit run dashboard.py
